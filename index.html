How I Build

My agentic AI pattern is structured, testable, and recruiter-readable.

01 Understand the domain

I start with the real decision: patient safety, driving risk, neighborhood planning, or storm protection.

02 Design the workflow

I define state, routing choices, tools, evidence sources, fallback paths, and final output structure.

03 Build the agent

I use Python, LangGraph, LangChain, Ollama, MCP-style tools, Pydantic, Streamlit, and tests.

04 Make it explainable

I show traces, evidence, evaluation, limitations, and safety checks so the output feels trustworthy.
