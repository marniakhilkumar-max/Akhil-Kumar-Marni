# Resume Bullets for Agentic AI Portfolio

## RoadSage AI

- Built a production-style autonomous driving safety reasoning agent using LangGraph, LangChain, Ollama, and MCP-style tools, enabling dynamic routing across scenario parsing, hazard detection, evidence retrieval, safety evaluation, and final recommendations.
- Designed a stateless agent workflow where every run is derived from the current driving scenario and configuration, with no hidden session memory or cross-user state.
- Implemented router validation, max-step loop protection, deterministic fallback behavior, Pydantic state models, trace timelines, and Pytest coverage to make the system reliable beyond a simple LLM demo.

- Exposed focused medical utilities through a lightweight MCP server and reused them inside the agent workflow for structured, testable tool execution.
- Added safety safeguards including urgent red-flag escalation, grounded patient-facing answers, medical disclaimers, fallback execution, and tests for router validation, graph completion, MCP tools, and LLM failure paths.
